# Bot discord giveaway
> Un simple bot discord fait avec discord.js


## Installer

Cloner le répertoire Github

Commandes :
```
Cloner le répertoire
npm install
```

## Usage

Ouvrir config.json et remplacer

```
"prefix": "PREFIX",
"token": "TOKEN"
```

## Démarrer

`Lancer start bot.bat`
